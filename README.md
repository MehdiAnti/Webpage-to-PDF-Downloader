# Webpage to PDF Downloader

Save any webpage as a PDF using GitHub Actions.

## Usage

1. Go to **Actions** → **Save webpage as PDF**
2. Click **Run workflow**
3. Enter:
   - **URL** (required) — the webpage to convert
   - **Title** (optional) — custom filename (without .pdf)
4. Click **Run workflow**

The PDF will be saved to `download/` folder and committed to the repo.

## Example

URL: https://example.com

Title: my-page

→ Creates `download/my-page.pdf`
